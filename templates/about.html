Text Insight Extraction

Aspect Based Sentiment Analysis on text to extract the topic related sentiment. The Zomato in house grammar is developed to parse sentences semantically. The model heuristically make the N-cut in grammar rules. In each cut, the model probabilistically learns the relative order for each grammar,by minimizing the training loss.

Image Classification

Used Deep Learning model for predicting Image labels. A variant of Alexnet model is trained on Zomato in house moderated data.Also, developed the pipelines using Luigi for training, testing and prediction for Zomato photos.

Aesthatic Image Assessment

Developing Deep Learning Architecture for learning Image Aesthetics. The constraint of fixed size input to Deep Models compromises the aesthetics of original image. We incorporated the Adaptive Spatial Pooling technique to address this problem by fixing the layer size just before the Fully Connected Layers in Deep Models. This preserves the original image composition and enables the model to learn aesthetic features without any transformations.
